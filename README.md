# Epic Games Store

## Тема проекта

Консольное приложение, имитирующее простой магазин игр по типу Epic Games Store.

Программа позволяет добавлять игры, просматривать их список, покупать, запускать и ставить им оценки.

---

# Возможности программы

- Добавление новой игры
- Просмотр списка всех игр
- Покупка игры
- Запуск игры
- Оценка игры
- Выход из программы

Все игры хранятся в одном контейнере `vector<Game*>`, благодаря чему используется полиморфизм.

---

# Структура проекта

## Абстрактный класс

### Game

Базовый класс для всех игр.

Содержит общие данные:

- название игры
- ID
- цену

Также содержит общие методы:

- showInfo()
- getName()
- getId()
- getPrice()

---

## Дочерние классы

От класса `Game` наследуются:

- GTA
- Deltarune
- PapersPlease

Каждый класс переопределяет метод `showInfo()` и реализует собственное поведение через интерфейсы.

---

## Интерфейсы

В проекте используются три интерфейса.

### IBuyable

Позволяет покупать игру.

Метод:

```cpp
buy()
```

---

### IPlayable

Позволяет запускать игру.

Метод:

```cpp
play()
```

---

### IRateable

Позволяет поставить игре оценку.

Метод:

```cpp
rate(int score)
```

---

# Наследование

Все игровые классы наследуются от общего базового класса `Game`.

```
Game
│
├── GTA
├── Deltarune
└── PapersPlease
```

Благодаря этому все игры имеют одинаковую структуру, но собственную реализацию методов.

---

# Полиморфизм

Полиморфизм используется при хранении объектов разных типов в одном контейнере.

```cpp
vector<Game*> games;
```

При выводе информации вызывается один и тот же метод:

```cpp
game->showInfo();
```

Однако для каждой игры выполняется собственная версия метода.

---

# Использование интерфейсов

Каждая игра реализует три интерфейса:

- IBuyable
- IPlayable
- IRateable

Благодаря этому пользователь может выполнять различные действия с выбранной игрой:

- купить игру;
- запустить игру;
- поставить оценку.

---

# Инкапсуляция

Поля класса `Game` являются защищёнными (`protected`).

Доступ к ним осуществляется через специальные методы:

- getName()
- getId()
- getPrice()

Это позволяет безопасно работать с данными объекта.
